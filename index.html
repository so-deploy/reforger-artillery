Coordinate Input Guide:

XY Grid (3-digits): 050 080
Grid position, snaps to center by default, can be read from under the cursor on the map

XY Direct (4+ digits): 0500 0825
Precise position, can be acquired by using laser on the drone

Z Value: 050 080 120
Always direct, read from under the cursor on the map

Numpads: 050 080 120 94 (Numpad 94)
Use numpad grid subdivision to fine tune the position within the grid (can be used as many times as you like)
